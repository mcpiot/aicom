# 打包说明

## 重要提示

**打包前请先关闭正在运行的 exe 程序！**

如果之前的打包程序还在运行，会导致文件被占用，无法重新打包。

## 打包步骤

1. **关闭所有正在运行的 "串口调试工具.exe"**

2. **清理旧的打包文件（可选）**
   ```bash
   # 如果 dist 目录被占用，可以手动删除或重启后再删除
   ```

3. **执行打包命令**
   ```bash
   npm run build:win
   ```

4. **打包输出位置**
   - 安装程序：`release/串口调试工具 Setup 1.0.0.exe`
   - 便携版：`release/串口调试工具-1.0.0-portable.exe`
   - 未打包目录：`release/win-unpacked/`

## 如果遇到文件被占用错误

1. 打开任务管理器（Ctrl+Shift+Esc）
2. 找到并结束所有 "串口调试工具.exe" 进程
3. 或者重启电脑后再打包
